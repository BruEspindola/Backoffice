## :computer: Como rodar

1º Clone o repositorio
```bash
  https://github.com/BruEspindola/web.git
```
2º Instalar bibliotecas
```bash
  cd web
  code .
  ------
  npm i
  cd client
  npm i
```
3º Rodar backoffice
```bash
  cd .. 
    obs: esteja na raiz do projeto
    
  npm run dev
```
